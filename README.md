### Adding New Sections

1. Add HTML markup in `index.html`
2. Add styles in `styles.css`
3. Add interactivity in `script.js`

Edit the `galleryDataModal` object in `script.js`:

```javascript

    'event-id': {
        title: 'Event Title',

            'image-url-1.jpg',
```
